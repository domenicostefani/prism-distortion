Back to Homepage

This page contains audio demos fror the Prism multiband effect

All the rows first show the settings for the 8 frequency bands, which comprise per-band effect type (Fuzz, Overdrive, Distortion), Gain (0-5) and Tone (0-5)

Then each example contains 4 audio files: the top row shows the Dry (no Prism) and Wet (with Prism) signals, where an amplifier with cabinet simulation was used (Swanky Amp plugin, clean setting). The bottom row contains the DI input and straight pedal output (Dry DI,Wet DI).
